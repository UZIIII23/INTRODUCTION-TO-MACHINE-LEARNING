## Machine Learning Lab 6 : Linear Regression, Gradient Descent & Model Fitting

Overview:

This repository contains the implementation of fundamental machine learning concepts, including **Linear Regression**, **Gradient Descent**, and **Overfitting vs Underfitting**. Synthetic datasets based on engineering and physics principles are used to demonstrate regression modeling, optimization, and model evaluation.

Objectives:

* Generate synthetic datasets for regression analysis.
* Implement Linear Regression using Scikit-learn.
* Understand Gradient Descent optimization.
* Visualize regression lines and loss curves.
* Compare underfitting, optimal fitting, and overfitting using polynomial regression.
* Evaluate regression models using standard performance metrics.

Experiments Included:

### **Part A: Linear Regression using Ohm's Law**

* Generate synthetic voltage-current data using Ohm's Law.
* Train a Linear Regression model.
* Estimate the resistance (slope).
* Visualize the regression line.
* Evaluate prediction accuracy.

### **Part B: Gradient Descent**

* Normalize input and output data.
* Implement Gradient Descent from scratch.
* Optimize slope and intercept iteratively.
* Visualize convergence and fitted regression line.
* Rescale parameters back to physical units.

### **Part C: Overfitting vs Underfitting**

* Generate projectile motion data.
* Fit polynomial regression models of different degrees.
* Compare Degree 1 (Underfitting), Degree 2 (Best Fit), and Degree 10 (Overfitting).
* Analyze training and testing errors.
* Visualize model behavior.

Evaluation Metrics:

The models are evaluated using:

* Mean Absolute Error (MAE)
* Root Mean Squared Error (RMSE)
* Mean Squared Error (MSE)
* R² Score
* Training Loss
* Testing Loss
