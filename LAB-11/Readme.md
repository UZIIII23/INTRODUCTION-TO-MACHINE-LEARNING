## Machine Learning Lab 11 : 1D CNN for Time Series Forecasting

Overview:

This repository contains the implementation of a One-Dimensional Convolutional Neural Network (1D CNN) for multivariate time series forecasting. The notebook demonstrates preprocessing, model development, training, and performance evaluation.
