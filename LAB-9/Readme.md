## Machine Learning Lab 9 : Image Augmentation and CNN Classification using Keras ImageDataGenerator

Overview:

This repository contains the implementation of **image classification using Convolutional Neural Networks (CNNs)** with **Image Augmentation** using Keras **ImageDataGenerator**. The experiment demonstrates how image augmentation improves model generalization, reduces overfitting, and enhances classification performance on a multiclass plant leaf disease dataset.

### **Image Augmentation and CNN Classification**

* Build a CNN model using Keras Sequential API.
* Apply image augmentation techniques including rotation, shifting, shearing, zooming, and horizontal flipping.
* Train the CNN using augmented training images.
* Validate the model using validation data.
* Save the best model automatically using ModelCheckpoint.
* Plot training and validation accuracy and loss curves.
* Evaluate the trained model on the test dataset.
* Generate a confusion matrix and classification report to assess model performance.
